# NUMBER SYSTEM

1. **NATURAL NUMBER** → all positive number from 1 to infinity.
2. **WHOLE NUMBER** → when we include 0 in natural numbers they become whole numbers.
3. **INTEGER** → all counting numbers 0 to infinity and -1 to negative infinity.
4. **EVEN AND ODD** → number divisible by 2 are even else odd.
5. **RATIONAL NUMBER** → a number which can be expressed in the form of p/q and have a finite decimal structure.
6. **IRRATIONAL NUMBER** → infinite decimal structure.
7. **PRIME NUMBER** → a number which is divisible by 1 and itself is a prime number.
8. **COMPOSITE NUMBER** → number which has more than 2 factors.
9. **FACTORS** → if number is 1 is dividing number 2 that means number 1 is a factor of number 2.
10. **CO-PRIME** → co-prime are pairs of number which has only one common factor 1.

## DIVISIBILITY RULE

- **DIVISIBILITY RULE OF 2** - if a number whose last digit is an even number i.e. 2,4,6,8 including 0
- **DIVISIBILITY RULE OF 3** - if the sum of its digit is divisible by 3.
- **DIVISIBILITY RULE OF 4** - if the last two digits of a number are divisible by 4.
- **DIVISIBILITY RULE OF 5** - numbers which last with digits, 0 to 5.
- **DIVISIBILITY RULE OF 6** - if a number is divisible by both 2 and 3.
- **DIVISIBILITY RULE OF 7** - 
    - remove the last digit and double it. 
    - subtract from the remaining number. 
    - if the number 0 or recognizable 2-digit multipe of 7?
    - then your number is multiple of 7 otherwise repeat the same process
- **DIVISIBILITY RULE OF 8** -  if the last 3 digits of a number are divisible by 8.
- **DIVISIBILITY RULE OF 9** - if the sum of all the digits is divisible by 9.
- **DIVISIBILITY RULE OF 11** - if the difference of the sum of alternative digits of a number is divisible by 11.
- **DIVISIBILITY RULE OF 13** - add four times of the last digit of the number to the remaining number and repeat the process until you get a two digit number and then check if it is divisible by 13 or not.
