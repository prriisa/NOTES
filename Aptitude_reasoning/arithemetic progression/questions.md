`ques` if the 8th term of an arithematic progression be zero, then the ratio of its 28th and 18th term will be ?

a(n) = a + (n-1)d

0 = a + (7)d
-a = 7d
a = -7d

28th term = a + (27)d
= -7d + 27d
= 20d

18th term = a + (17)d
= -7d + 17d
= 10d

a(28) : a(18) = 20d:10d = 2:1
