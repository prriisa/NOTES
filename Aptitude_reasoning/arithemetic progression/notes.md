# ARITHEMETIC PROGRESSION

THE PROCESS of moving gradually to a state or a position is known as progression.

and when we progress through numbers we call it arithemetic progression.

difference btw 2 alternate numbers is same then we call it as arithematic progression. where the terms have common difference.

series = 2, 4, 6, 8

initial term = 2
common difference = 2

`a(n) = a + (n-1)d`

where a = initial term.
n = nth term
d = common difference

---

`S(n) = n/2 (2a + n-1(d))`
this is the formulae to find the sum of AP.

to find the total number of element you must know =` n = [(l-a)/d] + 1`

where the l is last element, a is first element and d is the common difference which is used to find the number of terms in an AP.
