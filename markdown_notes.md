### For writing a simple paragraph you may direct write in the respective row.

'this text is an example of simple paragraph directly written on the screen without using any tags or other elements'

### For adding styles in your paragraph you may use :-

1. <ins>for bold</ins> : use '**' on the starting and ending of the text. For example:

        **hello**

2. <ins>for italic</ins> : use '_' on the starting and the ending of the text. For example:

        _hello_

3. <ins>for underline</ins> : Use the tag given below in the your paragraph.

        <ins>hello</ins>

4. <ins>for adding a quote</ins> : Use '>' in the starting of the paragraph

        > hello

5. <ins>for adding strick through</ins> : use '~~' in the startng and ending of text.

        ~~hello~~

### For adding links in your file you might use:-

1. Normally type your link in the file. it will become a clickable link. For example:
   
        https://github.com/prriisa

    result will be: https://github.com/prriisa


2. For making a clickable word which redirects to a link you give.

       [Priya's Github](https://github.com/prriisa)
    the result will be:
    [Priya's Github](https://github.com/prriisa)

3. for making a non clickable link just add a pair of backteck in it. For example:

        `https://github.com/prriisa`

    result will be: `https://github.com/prriisa`

### For making an undefined list:-

Use '-' before starting your point. as shown below:-

     - this is point no. 1
     - point no. 2
     - point no. 3

this will give a result:-

 - this is point no. 1
 - point no. 2
 - point no. 3

 ### For making a nested list :-

in the next line just press the tab key to make a block and then :

     - this is point no. 1
        - this is  nested point 1
        - this is nested point 2
     - point no. 2
     - point no. 3

result will be:-

 - this is point no. 1
    - this is  nested point 1
    - this is nested point 2
 - point no. 2
 - point no. 3

### For making a table

    |amit|rohit|
    |-----|-----|
    |18|21|

output will be:-

|amit|rohit|
|-----|-----|
|18|21|

for text allignment in any column just add ':' on that side for example:-

- |:----| => left alligned
- |:----:| => center alligned 
- |----:| => right alligned
